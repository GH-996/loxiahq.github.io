在 GitHub Pages 上部署 Hexo

Posted on 2023-08-16 In hexo Views:

本文主要描述了如何使用GitHub托管静态博客。

如果希望使用GitHub来生成一个静态资源服务站，那就需要用到GitHub Pages的知识点了。

新建git仓库

  1. 登录到GitHub ；
  2. 创建一个新的仓库，名字为username.github.io，username换成你在GitHub上的用户名。

设置私有域名

假如你有一个私有域名yourname.com，想要通过私有域名访问你的博客。

  • 需要先在域名服务商做解析，将yourname.com cname到yourname.github.io，我这里以dayone.loxia.cc为例。

  • 然后在博客文件夹的source目录下新建一个名为CNAME的无后缀文件，里面写上自己的域名：yourname.com。我这里以dayone.loxia.cc为例。

  • 此时运行hexo g重新生成静态文件，会发现public目录下面自动生成了CNAME文件。

  • 运行hexo d重新部署，稍等一会儿就可以通过访问自己的域名查看博客主页了。
